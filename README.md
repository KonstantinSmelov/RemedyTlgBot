## RemedyBot ver.0.2
>### Телеграм бот для учёта лекарственных средств в домашней аптечке.

Бот находится в процессе написания.  
Реализованы базовые функции (сохранение, редактирование, удаление).

***Используемый в текущей версии бота стек:***
- Telegram Bot API
- Spring Boot
- JPA
- H2DB
- Lombok
